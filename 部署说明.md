## 🚀 部署步骤（约10分钟）

### 第一步：注册 GitHub（已有可跳过）
1. 打开 https://github.com
2. 点击 Sign up 注册账号

### 第二步：上传代码到 GitHub
1. 登录 GitHub 后点击右上角 "+" → "New repository"
2. 仓库名填：`insurance-app`
3. 选择 Public，点击 Create repository
4. 点击页面上的 "uploading an existing file"
5. 把整个 insurance-app 文件夹里的所有文件拖进去
   ⚠️ 注意：api 文件夹也要一起上传！
6. 点击 Commit changes

### 第三步：注册 Vercel（免费）
1. 打开 https://vercel.com
2. 点击 Sign Up → 选择 "Continue with GitHub"（用GitHub账号登录）

### 第四步：部署项目
1. 登录Vercel后点击 "Add New Project"
2. 找到 insurance-app 仓库，点击 Import
3. 不需要改任何设置，直接点击 Deploy
4. 等待约1分钟，部署完成

### 第五步：配置 API Key（关键！）
1. 打开 https://console.anthropic.com 注册/登录
2. 左侧点击 "API Keys" → "Create Key"
3. 复制这串密钥（sk-ant-开头）
4. 回到 Vercel，进入你的项目
5. 点击顶部 "Settings" → 左侧 "Environment Variables"
6. Name 填：`ANTHROPIC_API_KEY`
7. Value 填：你复制的密钥
8. 点击 Save
9. 回到项目首页，点击 "Redeploy"

### 完成！
Vercel会给你一个网址，格式类似：
`https://insurance-app-xxx.vercel.app`

这个网址就是你的专属工具，电脑手机都能用！

## 📱 微信/飞书内嵌方法

### 微信公众号内嵌
1. 登录微信公众平台 https://mp.weixin.qq.com
2. 进入"自定义菜单"
3. 新建菜单项，类型选"跳转网页"
4. 填入你的 Vercel 网址
5. 保存并发布

### 飞书内嵌
1. 在飞书群中，点击 "+" 添加小组件
2. 选择"网页应用"或"快捷方式"
3. 填入你的 Vercel 网址
4. 即可在飞书中直接使用

## ❓ 常见问题

**Q: API Key 收费吗？**
A: Anthropic新账号赠送免费额度，日常使用约$5-10/月，每次分析约$0.01

**Q: 客户数据安全吗？**
A: 数据通过你自己的Vercel服务器中转，历史记录存在本地浏览器，不会外泄

**Q: 手机上能用吗？**
A: 完全可以，界面已适配手机屏幕
